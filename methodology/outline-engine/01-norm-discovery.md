# 01-norm-discovery

**Step 1：找法引擎（Norm Discovery）**

从案件事实出发，发现与锁定候选请求权基础条文。

## 一、找法的基本逻辑

找法不是"猜法条"，而是一套结构化的规范发现流程：

```
案件事实 → 提取法律关系对（谁→谁） → 识别诉请类型 → 定位规范类别 → 锁定具体条文
```

这对应外在结构的五级检索：当事人 → 诉请 → 规范类别 → 具体规范 → 检视程式。

## 二、从事实到法律关系对

### 2.1 当事人识别

从事实中提取全部当事人，确定其法律地位：

- 自然人 / 法人 / 非法人组织
- 特殊身份：消费者、经营者、用人单位、雇员、监护人、物权人、占有人等

### 2.2 法律关系矩阵

两两配对，判断哪些方向上可能存在请求权：

- A → B（正向请求）
- B → A（反向请求/抗辩/反诉）
- A → C / C → A（第三人关系）

判断标准：当事人之间是否存在以下连接点之一：

- 合同关系（含先合同、后合同）
- 物权归属争议
- 侵权行为
- 不当得利关系
- 无因管理关系
- 身份关系（婚姻家庭、继承）

## 三、从法律关系到诉请类型

### 3.1 诉讼类型判定

| 类型 | 特征 | 典型标志词 |
| --- | --- | --- |
| 给付之诉 | 要求作为/不作为/支付金钱 | "赔偿""返还""支付""履行""交付" |
| 确认之诉 | 确认法律关系存否 | "确认……有效/无效""确认……存在/不存在" |
| 形成之诉 | 变更/消灭法律关系 | "撤销""解除""变更" |
| 非讼程序 | 单方申请 | "申请实现""申请保护令" |

### 3.2 给付内容细分

给付之诉项下进一步区分：

- **原给付请求权**：合同约定的主给付/从给付义务（如交货、付款）
- **次给付请求权**：违约所生请求权（继续履行、损害赔偿、违约金）
- **物权请求权**：返还原物、排除妨害、消除危险
- **损害赔偿请求权**：侵权损害赔偿、缔约过失损害赔偿
- **返还请求权**：不当得利返还
- **费用偿还请求权**：无因管理费用偿还

## 四、从诉请到规范类别

按外在结构优先级排序，逐一排查：

1. **合同请求权**（特殊优先于一般）
   - 原合同给付请求权
   - 违约损害赔偿请求权
   - 合同解除后返还/赔偿请求权
2. **类似合同请求权**
   - 缔约过失请求权
   - 无权代理人责任
3. **无因管理请求权**
4. **物权请求权**
   - 所有物返还（第235条）
   - 排除妨害（第236条）
   - 消除危险（第236条）
   - 占有保护请求权（第462条）
5. **侵权损害赔偿请求权**
   - 一般侵权（第1165条第1款）
   - 过错推定侵权
   - 无过错侵权（特殊侵权）
6. **不当得利请求权**（兜底）
   - 给付不当得利（第985条）
   - 权益侵害不当得利（第985条）

关键判断规则：

- **有合同先检合同**：当事人间存在有效合同 → 优先穷尽合同请求权
- **物权优先于债权**：对物的支配力 → 物权请求权独立于侵权
- **不当得利是兜底**：其他请求权不能完全覆盖时才考虑
- **注意竞合与聚合**：同一事实可能同时支撑多个请求权

## 五、从规范类别到具体条文

### 5.1 法律数据库辅助找法

当确定了规范类别后，通过法律数据库精确定位条文：

**Step 5.1.1: 使用 `[SLOT-LEGAL-DB]` 的语义检索功能**

- 输入：案件事实的关键描述 + 规范类别关键词
- 输出：候选条文清单（含条文号+摘要）

**Step 5.1.2: 通过 `[SLOT-LEGAL-DB]` 获取全文**

对候选条文逐一加载全文。

**Step 5.1.3: 规范属性初判**

判断每个候选条文是否为"主要规范"（请求权基础）：

- 是：进入 Step 2 精确分类
- 否：标记为辅助规范/防御规范，待挂载

**Step 5.1.4: 配套司法解释强制检索**

对已确认的主要规范，必须检索其配套司法解释：

- 通过 `[SLOT-LEGAL-DB]` 检索"最高人民法院关于适用《……》的解释"
- 通过 `[SLOT-LEGAL-DB]` 按主要规范条文号检索相关解释条文
- 将检索到的司法解释条文按规范属性分类后挂载到检视程式相应位置

常见配套司法解释（非穷尽）：

- 侵权类：《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》
- 合同类：《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》
- 物权类：《最高人民法院关于适用〈中华人民共和国民法典〉物权编的解释（一）》
- 担保类：《最高人民法院关于适用〈中华人民共和国民法典〉有关担保制度的解释》
- 时效类：《最高人民法院关于审理民事案件适用诉讼时效制度若干问题的规定》

### 5.2 典型请求权基础条文速查

以下为最常用的请求权基础条文（主要规范），供快速锁定：

**合同请求权：**

- 原给付：合同条款 + 第509条第1款 / 各有名合同义务条款
- 违约赔偿：第577条（违约责任总则）/ 第583条（并存损害赔偿）/ 第578条（预期违约）
- 强制履行：第577条前半段 / 第580条（不能强制履行的排除）
- 违约金：第585条
- 合同解除后：第566条
- 缔约过失：第500条 / 第501条

**无因管理：** 第979条第1款 / 第980条

**物权请求权：** 第235条（返还）/ 第236条（排除妨害/消除危险）/ 第462条（占有保护）

**侵权请求权：**

- 一般侵权：第1165条第1款
- 过错推定：第1165条第2款 + 各特殊规定
- 产品责任：第1202条 / 第1203条
- 机动车交通事故：第1208条
- 环境侵权：第1229条
- 高度危险：第1236条
- 动物损害：第1245条 / 第1246条 / 第1247条
- 建筑物/物件：第1253条 / 第1252条 / 第1254条

**不当得利：** 第985条

**担保请求权：** 第386条（担保物权总则）/ 第410条（抵押权实现）/ 第436条（质权实现）/ 第447条（留置权实现）

## 六、找法的常见陷阱

- **遗漏竞合请求权**：违约与侵权竞合（第186条）——不能只检其一
- **忽视第三人请求权**：合同相对性的例外（如第522条第三人利益合同）
- **混淆诉讼类型**：解除合同是形成之诉（规范基础），解除后的返还是给付之诉（请求权基础）
- **遗漏附随义务**：第509条第2款的保护、通知、协助义务可能产生独立请求权
- **忽视特别法**：消费者权益保护法、食品安全法、产品质量法中的特别请求权基础

## 七、输出格式

找法完成后输出候选请求权基础清单：

```
法律关系对 [A → B]：
  诉请1：[具体诉请内容]
    候选请求权基础：
      ① 第XXX条（[条旨]）——[规范类别初判：合同/侵权/...]
      ② 第YYY条（[条旨]）——[竞合/聚合关系说明]
  诉请2：...

法律关系对 [B → A]：
  ...

聚合说明：诉请1与诉请2可同时主张
竞合说明：候选①与候选②基于同一事实，择一行使
```
