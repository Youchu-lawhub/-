# inner-structure

鉴定式案例分析的内在结构（三层四步）

鉴定式分析的"内在结构"指：从一个具体的设问出发，沿着请求权或抗辩权的检视，按统一的逻辑层级展开的内部章节顺序。

## 一、三层

任一具体请求权（或抗辩权）的检视，分为相互递进的三层：

1. 请求权（或抗辩权）已产生
2. 请求权（或抗辩权）未消灭
3. 请求权（或抗辩权）可行使

只有上一层成立，才进入下一层；任一层不成立，整个请求权（或抗辩权）即被否定。

## 二、四步（每一层内部）

每一层内部按"权利产生 → 权利消灭 → 权利障碍 / 权利行使阻却"的顺序展开，对照式四步走：

**第一步：权利产生要件（构成要件）**

逐一检视该请求权或抗辩权的全部构成要件是否在本案中成就。

每个要件下：先客观陈述要件含义，再涵摄案件事实，得出小结论。

要件含义不清楚 → 进入"法律解释"子流程（参 methodology/legal-interpretation-toolkit.md）。

**第二步：产生抗辩**

针对"权利产生"层面的反对事由（如行为能力缺陷、意思表示瑕疵、违法、违背公序良俗等），检视是否存在能阻断权利产生的事实。

**第三步：消灭事由**

进入"未消灭"层面：检视清偿、抵销、提存、混同、免除、解除、撤销、时效届满（仅在未消灭层）等使权利已经消灭的事由是否发生。

**第四步：停止性抗辩 / 行使障碍**

进入"可行使"层面：检视先履行抗辩、不安抗辩、同时履行抗辩、保证人先诉抗辩、留置、诉讼时效抗辩（如以抗辩形式提出）等使权利暂时不能行使的事由。

## 三、抗辩—再抗辩链

当被告提出抗辩，原告可提出再抗辩（消灭被告抗辩的反向事由）；如果再抗辩成立，被告可再提再再抗辩……如此递进，直至无新事由可主张为止。

写作时按"主张 → 抗辩 → 再抗辩 → 再再抗辩"的递进结构，每一层均完成"产生—消灭—行使"的三层四步检视。

## 四、否认 vs 抗辩的区分

- **否认**：直接否定原告主张事实的真实性。否认者无须主张新的事实，仅否认对方之主张为真。
- **抗辩**：承认原告主张事实，但主张存在另一项足以排除/消灭/阻却原告权利的新事实。

二者在举证责任和写作位置上完全不同：

- 否认 → 在该构成要件涵摄环节直接得出"要件未成就"的结论；
- 抗辩 → 进入该层的抗辩 / 消灭 / 行使阻却环节，由被告承担举证。

## 五、内部排序的固定顺序（写作友好型）

为使报告排版稳定、读者可预测，内部检视顺序固定为：

```
A. 第一层：请求权已产生
   A.1 责任成立要件（产生要件）
       A.1.1 要件一
       A.1.2 要件二
       ...
   A.2 责任范围
   A.3 产生抗辩
B. 第二层：请求权未消灭
   B.1 消灭事由检视
C. 第三层：请求权可行使
   C.1 停止性抗辩检视
```

确认之诉、形成之诉、非讼程序的检视无须严格走完三层，只需根据规范基础给出该确认/形成/非讼之要件即可。

## 六、抗辩权的检视

抗辩权（如时效抗辩、同时履行抗辩、不安抗辩、先诉抗辩等）本身亦按三层四步检视：

1. 抗辩权已产生（构成要件齐备）；
2. 抗辩权未消灭（如不再具备同时履行的客观情形）；
3. 抗辩权可行使（如已经按程序提出主张）。
