# examination-sequence

鉴定式案例分析的外在结构（五级检索顺序）

"外在结构"指：当案件涉及多个当事人和多项可能的请求权时，整个报告的章节顺序——按何种逻辑把各项请求权的检视依次展开。

## 一、五级检索顺序

按以下五个维度递进，逐一锁定本案需检视的全部请求权与规范基础：

**第一级：双方关系（"谁向谁"）**

列出本案全部当事人。

两两建立请求权可能性矩阵，确定本案存在哪几对法律关系（如：A 对 B、B 对 A、A 对 C、C 对 A 等）。

每一对关系都可能产生独立的请求权检视。

**第二级：诉讼请求（"主张什么"）**

对每一对法律关系，识别主张方所提出的具体诉请类型：

- 给付之诉：要求作为/不作为/支付金钱/移转所有权/履行义务等；
- 确认之诉：要求确认某法律关系的存在或不存在；
- 形成之诉：要求变更或消灭某法律关系（如撤销、解除）；
- 非讼程序：如申请实现担保物权、申请人身安全保护令。

每一项诉请独立列章。

**第三级：规范类别（"基于什么类型的请求权"）**

按以下 18 类规范进行预选（覆盖民法全部请求权与规范基础）：

| 序号 | 规范类别 | 典型范例 |
| --- | --- | --- |
| 1 | 合同请求权 | 买卖、租赁、承揽、运输等给付义务条款 |
| 2 | 准合同请求权 | 无因管理、不当得利 |
| 3 | 物权请求权 | 返还原物、排除妨害、消除危险、恢复原状 |
| 4 | 占有保护请求权 | 占有物返还、占有排除妨害、占有消除危险、占有损害赔偿 |
| 5 | 人格权请求权 | 停止侵害、排除妨碍、消除影响、赔礼道歉 |
| 6 | 知识产权请求权 | 停止侵害、销毁、赔偿损失（与人格权请求权类似） |
| 7 | 侵权损害赔偿请求权 | 一般侵权、特殊侵权（产品/医疗/环境/机动车交通事故等） |
| 8 | 缔约过失损害赔偿请求权 | 违反先合同义务的损害赔偿 |
| 9 | 后合同义务请求权 | 履行后合同义务、违反后合同义务的损害赔偿 |
| 10 | 婚姻家庭请求权 | 抚养费、扶养费、赡养费、探望权等 |
| 11 | 继承请求权 | 法定继承、遗嘱继承、遗产管理等 |
| 12 | 债的保全 | 代位权、撤销权 |
| 13 | 债权债务转移 | 债权转让、债务承担、债权债务概括转移、债务加入 |
| 14 | 担保物权请求权 | 抵押、质押、留置 |
| 15 | 确认之诉规范基础 | 合同效力确认、所有权确认、用益/担保物权确认 |
| 16 | 形成之诉规范基础 | 合同撤销、合同解除、婚姻撤销、收养撤销 |
| 17 | 非讼程序规范基础 | 申请实现担保物权、申请人身安全保护令 |
| 18 | 程序性请求 | 管辖、保全、回避 |

**第四级：具体规范（"基于哪一条文"）**

在第三级类别中定位具体的请求权基础或规范基础（精确到条、款、项）。本步骤通过解题大纲生成引擎（methodology/outline-engine/）定位具体条文并构建检视程式；如用户已注册请求权基础类知识库，可优先参考。

**第五级：检视程式（"按什么步骤检验"）**

对每一具体规范基础，依"内在结构（三层四步）"展开实质检视。这一级是写作主体。

## 二、检索顺序的优先级

当一个案件涉及多个请求权时，按以下顺序排序：

1. **当事人优先级**：以原告所主张的请求权为出发点，依"原告 → 被告 → 第三人"顺序展开。
2. **基础原因优先级**：合同请求权 → 准合同请求权 → 物权请求权 → 占有保护请求权 → 人格权请求权 → 侵权损害赔偿请求权 → 不当得利返还请求权（兜底）。

   这一顺序的依据是"特殊优先于一般"——合同关系是当事人自愿安排，应当首先穷尽；物权请求权基于物上支配力，优先于侵权请求权；不当得利兜底。
3. **聚合 vs 竞合**：
   - 聚合的请求权应分别独立检视（双方都进入正文）；
   - 竞合的请求权应分别独立检视，最后在结论部分说明"原告可在 X 与 Y 之间择一行使"。

## 三、与内在结构的对接

外在结构决定"写哪几个章节"；内在结构（参 inner-structure.md）决定"每个章节内部怎么写"。五级检索完成后，以其结果作为预选清单提交用户确认（详见 workflow/2a-preselection.md），确认后进入大纲与正文写作阶段。
