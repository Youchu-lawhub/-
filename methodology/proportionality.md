# proportionality

论证深度的比例原则

## 一、何为比例原则

不是所有要件都需要详尽论证。论证深度应与争议的强弱、法律意义的大小成比例：

- **明显成立的要件**：用一句裁判式陈述简短带过；
- **可能争议的要件**：进入鉴定式三段论（大前提→小前提→结论），但篇幅适度；
- **核心争议要件**：完整展开法律解释、典型案件类型化、对反对说回应。

## 二、判断要件深度的标准

针对每个构成要件，逐一回答以下问题：

**本案事实是否落入要件文义的核心射程？**

- 是 → 一句裁判式陈述（如"本案存在书面买卖合同，符合合同法第 X 条之合同关系要件"）。
- 否 → 至少进入鉴定式三段论。

**理性一般人能否准确把握该要件的含义？**

- 是 → 直接涵摄。
- 否 → 进入法律解释流程（参见 legal-interpretation-toolkit.md）。

**本案的事实在该要件上是否存在双方争议？**

- 否 → 简短带过。
- 是 → 完整展开，含正反双方论点的回应。

**该要件结论是否决定整个请求权之成立？**

- 是 → 核心要件，必须深入。
- 否（仅影响非核心结论）→ 适度展开。

## 三、写作语言的两种模式

### 鉴定式（Gutachten）

用于争议要件、需要解释的要件、事实涵摄存在不确定的要件：

- **大前提**：陈述法律规范及构成要件，假设语气表达；
- **小前提**：陈述案件事实并涵摄；
- **结论**：陈述语气（确定语气）。

语言模板：

```
若 A 主张 X 请求权，须满足要件 a₁、a₂、a₃。
要件 a₁ 是指…… [法律解释，如有必要]
本案中…… [事实涵摄]
故要件 a₁ 满足 / 不满足。
[继续要件 a₂、a₃……]
综上，A 享有 / 不享有 X 请求权。
```

### 裁判式（Urteilsstil）

用于明显成立或明显不成立的要件、辅助性的程序事项、显而易见的事实判断：

直接给出结论 + 简要说明理由。

语言模板：

> 本案中，A 与 B 已签订书面买卖合同，合同关系成立，符合民法典第595条之要件。

## 四、混合写作

一份合格的鉴定式报告并非全篇鉴定式。核心争议用鉴定式，非核心要件用裁判式——这是比例原则在写作语言层面的表现。错误模式：

- **过度鉴定式**：每个要件都"假设语气—涵摄—结论"，全文冗长枯燥；
- **过度裁判式**：核心争议要件也只一句话带过，论证不充分。

## 五、对反对说的回应深度

- **次要反对说**：一句话回应即可（"或有观点认为……此说与本案规范目的相悖，故不采"）；
- **主要反对说**：需展开"前提—理由—我方反论"的完整结构；
- **判例采纳的反对说**：必须展开并说明本案与该判例的事实差异，或主张应予限缩。

## 六、自检清单

提交报告前自查：

- [ ] 是否对每个核心要件都做了"明显 / 争议 / 核心争议"的分类？
- [ ] 各要件的论证深度是否与其重要性相匹配？
- [ ] 全文是否避免了"全篇鉴定式"或"全篇裁判式"的极端？
- [ ] 主要反对说是否得到与其分量匹配的回应？
- [ ] 是否避免了"显然如此""不言自明"等回避性表述？
